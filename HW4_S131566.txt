(a)-1 예제 프로그램의 정적인 기하 물체 또는 기타 정적 물체를 사용하여 재미있게 건물 내부를 구성하라. - 구현(O)
	
- (30, 85, 9.5)에 소 배치
- (73, 105, 30)에 액자 배치
- (73, 105, 0)에 의자 배치
- (188, 25, 10)에 주전자 배치
- (188, 25, 0)에 테이블 배치


(a)-2 호랑이를 건물 내에서 올바른 경로를 통하여 움직이도록 하라. - 구현(O)

- 호랑이를 건물내 일정한 경로를 반복해서 움직이도록 구현


(a)-3 계층적 모델링 기법이 적용된 동적 물체를 배치하라 - 구현(O)

- 계층적 모델링이 적용된 자동차가 건물 주변을 돌게 끔 배치


(b)-1 마우스 또는 키보드 등의 입력 장치에 반응하여 주 카메라가 카메라의 세 기본축 방향으로 이동하도록 하라 - 구현(O)

** 해당 조작은 주 카메라에만 적용됨

- 키보드의 왼쪽 화살표 버튼을 누르면 카메라의 uaxis 반대 방향(왼쪽)으로 이동
- 키보드의 오른쪽 화살표 버튼을 누르면 카메라의 uaxis 방향(오른쪽)으로 이동

- 키보드의 아래 화살표 버튼을 누르면 카메라의 vaxis 반대 방향(아래)로 이동
- 키보드의 위 화살표 버튼을 누르면 카메라의 vaxis 방향(위)로 이동

- 키보드의 PAGE_DOWN 버튼을 누르면 카메라의 naxis 방향(앞)으로 이동
- 키보드의 PAGE_UP 버튼을 누르면 카메라의 naxis 반대 방향(뒤)로 이동

(b)-2 마우스 또는 키보드 등의입력 장치에 반응하여 주 카메라가 카메라의 세 기본축 둘레로 회전하도록 하라. - 구현(O)

** 해당 조작은 main view 카메라에 적용됨

- 키보드의 q 버튼을 누르면 카메라의 vaxis 방향으로 회전
- 키보드의 w 버튼을 누르면 카메라의 vaxis 반대 방향으로 회전

- 키보드의 a 버튼을 누르면 카메라의 uaxis 방향으로 회전
- 키보드의 s 버튼을 누르면 카메라의 uaxis 반대 방향으로 회전

- 키보드의 z 버튼을 누르면 카메라의 naxis 방향으로 회전
- 키보드의 x 버튼을 누르면 카메라의 naxis 반대 방향으로 회전


(b)-3 마우스 또는 키보드 등의입력 장치에 반응하여 주 카메라가 화면의 내용을 확대 또는 축소를 할 수 있도록 하라. - 구현(O)

** 해당 조작은 main view 카메라에 적용됨

- 키보드의 v 버튼을 누르면 축소
- 키보드의 b 버튼을 누르면 확대


(b)-4 주 카메라 외에 정적인 CCTYV 카메라 기능을 구현하라. - 구현(O)

- (190, 120, 20)에서 (200, 120, 19)를 바라보는 방향으로 cctv1 구현, p 버튼으로 해당 뷰로 전환 가능
- (155, 75, 10)에서 (0, 75, 10)을 바라보는 방향으로 cctv2 구현, o 버튼으로 해당 뷰로 전환 가능
- (200, 100, 50)에서 (200, 100, 0)을 바라보는 방향으로 cctv3 구현 i 버튼으로 해당 뷰로 전환 가능

- u 버튼으로 월드 뷰 (주 카메라)로 전환

** cctv는 카메라 이동을 제외한 모든 움직임 (회전 및 확대) 가능


(b)-5 한 대의 동적인 CCTV를 구현하라. - 구현(O)

- 호랑이의 시점으로 동적인 cctv를 구현, t 버튼으로 해당 뷰로 전환 가능


(b)-6 정면도-측면도-상면도에 해당하는 카메라 구현 - 구현(O)
- 뷰포트를 나누어 정면도, 측면도, 상면도를 구현

(b)-7 주 카메라 외에 자신이 구현한 cctv 카메라 또는 정면도-측면도-상면도에 해당하는 카메라에 의한 화면의 내용에 주 카메라의 뷰잉 볼륨을 도시  - 구현(O)
- 카메라의 뷰잉볼륨을 뷰포트로 나누어져 있는 정면도, 측면도, 상면도에서 관찰 가능
- y 버튼을 통해 뷰잉볼륨을 껐다켰다 할수 있음.
