그래픽스 과제 4 README

과제 2에서 key

- KEY_UP : 카메라가 바라보는 방향 기준으로 위로 이동
- KEY _DOWN : 카메라가 바라보는 방향 기준으로 아래로 이동
- KEY _LEFT : 카메라가 바라보는 방향 기준으로 왼쪽으로 이동
- KEY _RIGHT : 카메라가 바라보는 방향 기준으로 오른쪽으로 이동
- PAGE_UP : 카메라가 바라보는 방향 기준으로 앞으로 이동
- PAGE_DOWN : 카메라가 바라보는 방향 기준으로 뒤로 이동

- q : 카메라의 x  축으로 시계방향 회전
- w : 카메라의 x  축으로 반시계방향 회전
- a : 카메라의 y  축으로 시계방향 회전
- s : 카메라의 y  축으로 반시계방향 회전
- z : 카메라의 z  축으로 시계방향 회전
- x : 카메라의 z  축으로 반시계방향 회전

- v : 카메라의 뷰 각도를 넓힘
- b : 카메라의 뷰 각도를 좁힘

- p : 메인뷰를 cctv1으로 변경
- o : 메인뷰를 cctv2으로 변경
- i : 메인뷰를 cctv3으로 변경
- u : 메인뷰를 world_camera 로 변경

- y : 카메라 뷰 볼륨 on/off


[기본 쉐이딩 효과]

(a) : 기본 쉐이더로는 Phong Shader가 적용되어 있음
(b) : 사용자가 오른쪽 쉬프트 키와 마우스 오른쪽 버튼을 누르면 Gouraud Shader 적용
(c) : 위 두 키를 동시에 때면 다시 Phong Shader 적용

[조명효과]

** 조명은 총 6개로
0번 조명 : 카메라로 부터 나가는 조명
1번 조명 : 건물 내부에 설치된 물체 light로 부터 나가는 빛 (모든 방향으로 빛을 방출)
2번 조명 : 건물 내부에 설치된 물체 light로 부터 나가는 빛 (빨간색 조명)
3번 조명 : 건물 내부에 설치된 물체 light로 부터 나가는 빛 (초록색 조명)
4번 조명 : 건물 내부에 설치된 물체 light로 부터 나가는 빛 (파란색 조명)
5번 조명 : 건물 내부에 설치된 물체 light로 부터 나가는 빛 (스포트라이트)

** 각각의 조명은 해당 조명 번호의 숫자키로 on/off 가능

(a) : F1 키로 블라인드 조명 효과 on/off
       F2 키로 블라인드 조명의 원을 증가
       F3 키로 블라인드 조명의 원을 감소

(b) : F4키로 건물내부의 모든 조명이 랜덤한 색과 방향, 블라인드 효과로 빛을 냄

	
[기하 효과]

(a) : F5 키로 건물 내부에 스크린을 on/off
       F6 키로 스크린의 격자를 증가
       F7 키로 스크린의 격자를 감소
